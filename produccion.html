Producción


El proyecto se lanzó en fase Alfa en internet el 24 de mayo de 2024. Actualmente, sigue en fase Alfa. La decisión de lanzarlo se tomó con la intención de ofrecer un servicio más allá de un servidor local. Publicarlo en internet representó un desafío significativo; inicialmente, queríamos manejar el servidor en una máquina propia, pero la configuración de Django y los servidores web resultó ser compleja. Sin embargo, al descubrir render.com, todo se simplificó. A continuación, se detalla la estructura que tenemos para el proyecto en producción.

Render.com


Render.com es una plataforma de alojamiento en la nube que permite a desarrolladores y empresas desplegar y gestionar aplicaciones, sitios web, bases de datos, y otros servicios en la nube de manera sencilla. Render se enfoca en proporcionar una experiencia de despliegue y gestión de infraestructuras simplificada, similar a la de servicios como Heroku.



Deploy


Preparar el despliegue de una aplicación en internet requiere una cuidadosa revisión de la documentación disponible. En primer lugar, el proyecto se gestiona desde un repositorio local, donde cada commit y push a GitHub desencadena automáticamente el despliegue en Render.com. Durante este proceso, se ha realizado una importante configuración: la migración de la base de datos desde SQLite a PostgreSQL.

Además de alojar la aplicación, Render.com ofrece servicios de bases de datos, incluyendo PostgreSQL. Se ha configurado para que la aplicación apunte a esta nueva base de datos PostgreSQL. Para gestionar los datos, se ha utilizado un cliente local (Ubuntu) para conectarse a la base de datos. Actualmente, estamos en el proceso de implementar MongoDB. Se ha creado un servidor en Mongo Atlas que servirá como base de datos en la nube para la aplicación web.



Estado


Actualmente, el proyecto sigue en línea y aún no se ha tomado la decisión de continuar trabajando en él o no. Esto implica un costo de tiempo y posiblemente un costo económico si se decide seguir adelante. Cada miembro del grupo tomará su propia decisión al respecto. A día de hoy, mayo de 2024, la página web sigue disponible para visitar.

Dado que se trata de planes gratuitos, no se espera un rendimiento óptimo en cuanto a velocidad de carga u otros aspectos técnicos. Sin embargo, a pesar de esto, la plataforma es completamente funcional. Normalmente, habrá un administrador encargado de activar usuarios y realizar el mantenimiento mínimo necesario para garantizar una buena experiencia para los usuarios.
